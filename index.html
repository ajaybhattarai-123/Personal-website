Real-Time Bidirectional Traffic Analysis

Methods: YOLOv8 (yolov8l.pt), ByteTrack, Computer Vision

Problem: Addressing the need for automated, error-free monitoring in traffic management and planning to replace manual counting methods.

  • Achieved high accuracy in multi-class vehicle classification (Car, Bus, Motorbike, Truck).
  • Enabled robust real-time tracking across frames using ByteTrack.
  • Automated data export to Excel (.xlsx) for seamless downstream analysis.
PyTorch YOLOv8 ByteTrack OpenCV Pandas

Automated Concrete Crack Detection System

Methods: Custom CNN, PyTorch, Deep Learning

Problem: Provides a fast, non-destructive, and objective method to classify surface health for maintenance prioritization.

Engineered a custom Convolutional Neural Network (CNN) using PyTorch, incorporating Dropout and Batch Normalization layers to ensure robust performance on structural surface images.

  • Achieved >98% accuracy on the test set for binary classification.
  • Optimized for concrete and pavement surface images.
  • Significantly reduced inspection time compared to manual methods.
PyTorch Python NumPy Matplotlib CNN
